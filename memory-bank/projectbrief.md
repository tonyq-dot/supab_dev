# Project Brief: KPI System

Система учёта работ и расчёта KPI/бонусов для студии анимации (VFX/Houdini).

## Проблема
Excel-таблица с формулами расчёта бонусов неудобна для творческих сотрудников. Ручной ввод отвлекает от работы.

## Решение
Веб-приложение (React + Supabase), которое:
- Собирает данные о проделанных работах
- Автоматически считает баллы и премии
- Экспортирует отчёты в CSV/Excel для руководства

## Роли
- **Аниматор** — вносит работы, видит баллы и прогноз премии
- **Менеджер** — управляет проектами, настройками, просматривает/редактирует все данные

## Стек
React, Vite, TypeScript, Supabase (PostgreSQL, Auth, Storage)
